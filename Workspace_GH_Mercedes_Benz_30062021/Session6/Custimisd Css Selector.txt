Css Selector is also the position of the element in the DOm document

Css Selector are of two types:
1) Partial or relative Css Selector
2) Absolute or Full Css Selector - it always start with HTML tag

Customisation of Css Selector is needed when the Css Selector given by using the inspect element in the browser
opened by Selenium or by inspect element in the desktop version is not working. It gives NO SUCH ELEMENT 
EXCEPTION. For customisation , first create PARTIAL Css Selector and if partial xpath is also giving 
NO SUCH ELEMENT exception, then create FULL Css Selector

Format of Partial or relative xpath:

//*[@attribute present in HTML script = 'value of the attribute']

// --> All parent or all child tags
* --> Regular expression meaning any html tag
@ --> regular expression defining a specific attribute
/ --> immediate parent tag or immediate child tag

Format of Partial or relative CssSelector:

*[attribute present in HTML script = 'value of the attribute']
* --> Regular expression meaning any html tag
Single Space or > (grater than sign) --> immediate parent tag or immediate child tag
. --> sign used to represent CLASS attribute
# --> sign used to represent ID attribute
